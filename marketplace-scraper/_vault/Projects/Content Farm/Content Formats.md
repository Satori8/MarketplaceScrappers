# Content Formats

## X / Twitter

Formats:

- short observation
- 5-9 tweet thread
- tool comparison
- "people are complaining about X"
- "new tool solves Y"
- weekly digest

Rules:

- one idea per post
- strong first line
- no generic AI tone
- avoid link-only posts
- use source insight, not source copy

## Telegram

Formats:

- compact digest
- tool drop
- explanation post
- curated links
- mini-case study

Rules:

- more direct than X
- can include links more naturally
- good place for affiliate/service funnel

## LinkedIn

Formats:

- operator insight
- mini case study
- workflow breakdown
- B2B pain point
- automation lesson

Rules:

- less edgy
- more credibility
- avoid obvious content-farm smell

## Short Video

Formats:

- problem -> tool -> result
- 3 tools for X
- mistake people make
- trend explained in 45 seconds
- Reddit complaint -> solution

Useful for:

- YouTube Shorts
- TikTok
- Instagram Reels
- Facebook Reels

## Newsletter / Blog

Formats:

- weekly trend report
- "top 10 problems people discussed this week"
- buyer guide
- tool comparison
- niche market intelligence
