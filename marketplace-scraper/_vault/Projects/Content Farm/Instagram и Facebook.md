# Instagram и Facebook

## Вывод

Instagram и Facebook можно подключать, но не как первый backend автопостинга.

Лучше использовать их так:

```text
Content Farm -> draft -> approval -> Meta Business Suite / official API / scheduler
```

Не использовать:

- браузерных ботов;
- автолайки;
- автоподписки;
- авто-комментарии;
- массовый постинг одинакового контента;
- обход ограничений API.

## Instagram

Подходит для:

- Reels;
- каруселей;
- коротких объясняющих видео;
- визуальных подборок инструментов;
- "3 идеи / 5 инструментов / 1 тренд".

Слабее подходит для:

- длинных аналитических текстов;
- частого линк-постинга;
- полностью автоматического эксперимента без визуального стандарта.

## Facebook

Подходит для:

- Pages;
- Reels;
- постов с ссылками;
- групп вокруг темы;
- локальной украинской/русскоязычной аудитории 25+.

Осторожно:

- автопостинг в группы сильно рискованнее и технически ограниченнее;
- одинаковые посты в много мест выглядят как спам;
- лучше строить Page + community, чем массово разбрасывать ссылки.

## Что можно автоматизировать безопаснее

- подготовку drafts;
- ресайз/форматирование изображений;
- генерацию captions;
- календарь публикаций;
- планирование через Meta Business Suite;
- публикацию через официальные API/одобренные schedulers;
- сбор метрик;
- отчёт в Obsidian.

## Что нельзя делать

- имитировать человека в интерфейсе Meta;
- автоматически лайкать/подписываться/комментировать;
- парсить закрытые данные;
- постить один и тот же текст массово;
- скрывать рекламный/affiliate характер CTA.

## Практический план подключения

### Этап 1

- [ ] Делать drafts для Instagram/Facebook вручную из общей очереди.
- [ ] Публиковать через Meta Business Suite.
- [ ] Смотреть, какие форматы дают сохранения/комментарии.

### Этап 2

- [ ] Создать визуальный шаблон для Reels/каруселей.
- [ ] Автоматизировать подготовку captions и сценариев.
- [ ] Автоматизировать экспорт в папку `approved`.

### Этап 3

- [ ] Подключать официальный API или проверенный scheduler.
- [ ] Оставить human approval.
- [ ] Ограничить частоту публикаций.

## Форматы для Meta

### Instagram Reels

- "На Reddit обсуждают проблему X"
- "3 AI-инструмента для малого бизнеса"
- "Западный тренд, который можно применить у нас"
- "Идея микросервиса за 30 секунд"

### Instagram Carousel

- список инструментов;
- пошаговая схема;
- сравнение сервисов;
- "проблема -> решение -> CTA".

### Facebook Page

- расширенный пост;
- ссылка на дайджест;
- короткое видео;
- обсуждение с вопросом в конце.

### Facebook Group

Только если есть своя группа и понятная модерация.

Не строить MVP на автопостинге в чужие группы.
