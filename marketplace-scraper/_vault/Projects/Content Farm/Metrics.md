# Metrics

## Source Metrics

- items collected per source
- useful items per source
- average score
- duplicate rate
- time-to-signal

## Content Metrics

- drafts generated
- drafts approved
- drafts rejected
- reason for rejection
- time from source to draft

## Platform Metrics

- impressions
- likes
- comments
- shares
- bookmarks/saves
- profile visits
- link clicks
- follows
- language split: RU vs UA performance
- format split: text vs carousel vs short video

## Business Metrics

- affiliate clicks
- affiliate conversions
- leads
- email subscribers
- service inquiries
- revenue per domain

## Feedback Rule

Do not optimize for likes only.

Prioritize:

1. saves/bookmarks
2. comments from target audience
3. clicks
4. leads
5. revenue
