## ФАЗА 5 — Кастомні модулі
> Пріоритет зверху вниз

- [ ] **Модуль імпорту з Excel/Google Sheets**
  - Ключ: артикул (дублі/порожні = помилка в звіті)
  - Галочки «що оновлювати» при імпорті
  - Preview перед запуском + звіт з помилками по рядках
  - Автооновлення: ніколи / щодня / щотижня / щомісяця
- [ ] **Розширені фільтри в адмінці**
  - За: категорією, типом, ціною, видимістю, наявністю, знижкою, брендом
  - Перегляд видалених позицій
- [ ] **Логіка самовивозу**
  - Прапорець «Можливий самовивіз» в картці товару
  - Прив'язка до конкретного складу
  - При оформленні — показувати лише прив'язані точки
- [ ] **Калькулятор плитки**
  - Знає м² в пачці з характеристик товару
  - Ціна за м², замовлення кратне пачці
  - Округлення вгору: 18 м² ÷ 1,44 = 12,5 → 13 пачок
- [ ] **Масові знижки**
  - Застосувати % знижки до вибірки (категорія / бренд / фільтр)
  - Відображення: ціна до/після + %
