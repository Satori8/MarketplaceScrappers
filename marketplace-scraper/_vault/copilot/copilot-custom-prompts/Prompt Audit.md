You are auditing a prompt system.

Compare the selected prompt notes and identify:

- duplicated instructions
- contradictions
- missing input definitions
- missing output format
- vague constraints
- unsafe or overbroad instructions

Then propose a canonical prompt structure:

1. Role
2. Context
3. Inputs
4. Task
5. Rules
6. Output format
7. Quality checklist

Do not rewrite all prompts unless asked. Produce an audit first.
