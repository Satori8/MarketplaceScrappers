# Scrappers

A collection of marketplace scrapers and data processing tools.

## Included Projects

- **Marketplace Scraper**: A robust Ukrainian marketplace scraping pipeline with GUI, database integration, and Excel export.

## Setup

1. Create a virtual environment:
   ```bash
   python -m venv .venv
   ```
2. Activate the virtual environment:
   - Windows: `.venv\Scripts\activate`
   - Linux/macOS: `source .venv/bin/activate`
3. Install dependencies:
   ```bash
   pip install -r marketplace-scraper/requirements.txt
   ```

## Usage

Run the main application:
```bash
python marketplace-scraper/main.py
```
